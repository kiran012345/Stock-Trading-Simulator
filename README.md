# Stock Trading Simulator (C++)

A console-based stock trading simulator built using C++. The application allows users to simulate stock trading activities such as buying and selling stocks, managing portfolios, and tracking transaction history.

## Features
- Multi-user trading accounts
- Buy and sell stock orders
- Portfolio management
- Transaction history tracking
- Order matching engine using priority queues
- Real-time stock price simulation

## Technologies Used
- C++
- Object Oriented Programming (OOP)
- STL (vector, map, priority_queue)

## How to Run

Compile the program:

g++ trading.cpp -o trading

Run the program:

./trading

## Concepts Demonstrated
- Data Structures
- Algorithm Design
- Heap-based priority queues
- Trading engine simulation
